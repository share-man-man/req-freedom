LOCAL FIXTURE · REQUEST LAB

Req Freedom 请求实验室

一个零依赖的静态页面，用来观察规则对浏览器请求和页面行为的影响。

页面脚本已就绪
使用方式：使用任意 Node 静态服务托管本目录，打开页面后按卡片中的规则示例配置插件；每次请求的 URL、状态、耗时和响应会显示在下方日志中。Header 改写请同时在 DevTools Network 中查看真实请求/响应头。
FETCH

基础请求 / 返回值 Mock

读取用户数据。可将这条请求直接 Mock 成任意状态码、响应头和 JSON 响应体。

/api/users.json
Mock 规则示例 
匹配：contains /api/users.json
状态：503
响应体：{"code":503,"message":"mocked"}
XHR

XHR / 返回值 Mock

使用传统 XMLHttpRequest 请求商品数据，专门验证 XHR 的 Mock 与延迟路径。

/api/products.json
Mock 规则示例 
匹配：contains /api/products.json
状态：200
响应体：{"source":"xhr-mock","items":[]}
DNR

重定向

请求 source 文件；重定向后，日志中的最终 URL 和响应内容应变为 target 文件。

/api/redirect-source.json → /api/redirect-target.json
重定向规则示例 
匹配：contains /api/redirect-source.json
目标：当前地址/api/redirect-target.json
DNR

参数注入

文件服务会忽略查询参数，但最终请求 URL 会在日志与 Network 面板中清楚显示。

/api/params.json?from=lab
参数规则示例 
匹配：contains /api/params.json
注入：debug=1，environment=local
DNR

请求 / 响应 Header

该请求带有初始 Header；页面会回显可读取的响应头，完整请求头请在 Network 中检查。

/api/headers.json
Header 规则示例 
请求头 set：X-Req-Freedom = enabled
响应头 set：X-Response-Patched = true
FETCH

网络限速 / 延迟

下载约 512 KB 的文本响应。可用 Slow 3G 或自定义延迟观察 loading、耗时和下载过程。

/api/slow-response.txt
限速规则示例 
匹配：contains /api/slow-response.txt
档位：Slow 3G
