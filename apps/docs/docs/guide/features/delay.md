# 延迟模拟

为命中的请求注入人为延迟，验证 loading 态、竞态与超时处理。

## 实现方式

与 Mock 相同走**页面补丁通道**：`fetch` 在真实请求前 `await` 延迟；XHR 将真实 `send` 推迟执行。

> 注意：仅对页面脚本发起的 fetch / XHR 生效。

## 规则字段

| 字段 | 说明 |
| --- | --- |
| `delayMs` | 延迟时长（毫秒） |

## 示例

模拟弱网下的慢接口：

- 匹配方式：`contains`
- 匹配模式：`example.com/api/list`
- 延迟：`3000` 毫秒

延迟规则可与 Mock 规则叠加：两者延迟时长相加后生效。
